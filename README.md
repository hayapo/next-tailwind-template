# Template for Next.JS + Tailwind CSS + TypeScript Project with ESlint + Prettier

## Package install

```bash
npm install
```

## How to run

```bash
yarn run dev
```

## Lint or Format code

### Linting

```bash
# next lint
yarn lint:next

# eslint
yarn lint:eslint
```

### Formatting

```bash
# format all codes with prettier
yarn format:all

# format codes under src with prettier
yarn format:src
```

## Features

- Next.js
- Typescript
- Tailwind CSS
- ESlint
- Prittier
